# Otto TuneLab

English | [简体中文](README_zh.md)

A toolbox to generate & optimaize project of TuneLab.

## Features

1. Extracts **pitch** from voice using [RMVPE](https://arxiv.org/abs/2306.15412v2).
2. Tranforms **TextGrid** generated by [SOFA](https://github.com/qiuqiao/SOFA) or [HubertFA](https://github.com/wolfgitpr/HubertFA) into **notes**. (Currently only supports Japanese)
3. Calculate **gender** and **breathiness** by taking the derivative of **pitch**.

## How to use

1. `git clone` this repo.
2. Create a python envirnment then `pip install -r requirements_cpu.txt`. (RMVPE is a small model that can run fast enough on CPU, you can modify the code and python dependencies to run it on GPU though)
3. Download `rmvpe.onnx` from [here](https://huggingface.co/lj1995/VoiceConversionWebUI/blob/main/rmvpe.onnx) and put in the root folder of this project.
3. Run `python webUI.py` then open `http://127.0.0.1:7860` in your browser.
